# Terra Time Input Field

TimeInputField is a component of terra-time-input that provides a container for rendering the related time inputs. TimeInputField includes displays for help text, error text, required or optional indication, and invalid states.

## Usage

```jsx
import TimeInputField from 'terra-time-input/lib/TimeInputField';
```
