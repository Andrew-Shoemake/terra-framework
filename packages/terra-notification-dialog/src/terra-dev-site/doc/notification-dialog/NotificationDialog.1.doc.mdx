# Terra Notification Dialog

The `terra-notification-dialog` component renders as a notification using the `terra-abstract-modal` to display a dialog over content. It is a common component that can be used to display a message to a user. When the user is presented with the dialog, it is paired with an acceptance and an optional rejection action to allow the user to correctly handle the notification.

The dialog will have the highest z-index of 9000, so it will display over all other content, including over modals.

There are five dialog variants build into `terra-notification-dialog`:

1. `alert`
2. `error`
3. `warning`
4. `success`
5. `info`

If one of these variants do not meet your needs, there is the option to customize the dialog and the icon it displays.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-notification-dialog`
